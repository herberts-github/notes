# 排序

## 排序实现

通过在 `SELECT` 末尾添加 `ORDER BY` 明确指定排列顺序

`ORDER BY` 书写的列名称成为 **排序键**

可使用存在表中、但不包含在 `SELECT` 子句的列，也可使用聚合函数

```sql
SELECT 
    column_name, ...
FROM
    table
ORDERY BY
    column_name [ASC / DESC]
```

- ASC：升序，默认
- DESC：降序
- 支持按多个字段排序

> 当同时指定多个排序键时，规则是：**优先使用左侧的键，如果该列存在相同值，再接着参考右侧的键**
> 
> **注意**：含有 `NULL` 的列作为排序键时，`NULL` 会在结果开头或末尾汇总显示

## 排序规则

- 数字类型（整数、浮点数）：按数字**大小**排序
- 日期时间：按**先后 / 早晚 / 大小** 顺序排序
- 字符类型：按字母的 ASCII 码顺序

## 列编号

在排序时不要使用列编号，会影响 SQL 代码阅读困难

列编号：`SELECT` 的列按照从左到右的顺序进行排列时所对应的编号（1，2，3...）

```sql
-- 列名
SELECT
    column_name1, column_name2,
FROM
    table 
ORDER BY
    column_name1 [ASC / DESC] , column_name2 [ASC / DESC];
    
-- 列编号
SELECT
    column_name1, column_name2,
FROM
    table
ORDER BY
    1 [ASC / DESC] , 2 [ASC / DESC];
```
